# New Portfolio

Fresh React + Vite rebuild of the portfolio frontend for Yaroslav Shapran.

## Run locally

```bash
npm install
npm run dev
```

Vite should run on `http://localhost:5173` unless that port is already in use.

## Build

```bash
npm run build
npm run preview
```

## Stack

- React 19
- Vite
- TypeScript
- Tailwind CSS
- DaisyUI
- React Router
- React Markdown

## Content

- portfolio data: `src/data/portfolio.ts`
- blog posts: `src/content/blog/*.md`
- public media: `public/blog`, `public/projects`
